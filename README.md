# Pro Plugin Marketplace

This repository is a Codex plugin marketplace that publishes `pro-plugin`, a ChatGPT Pro browser bridge for Codex.

## Install

```bash
codex plugin marketplace add xavierchoi/pro-plugin
codex plugin add pro-plugin@pro-plugin
```

Then start a new Codex thread so the plugin's skill and MCP tools are loaded.

## Plugin

See [plugins/pro-plugin/README.md](plugins/pro-plugin/README.md) for browser setup, Comet notes, 2FA handling, and E2E testing.
